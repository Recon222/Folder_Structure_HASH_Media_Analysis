üíæ Saving and Managing Templates

Saving a Template

  1. Enter a descriptive name in the "Template Name" field
  2. Click üíæ Save Template
  3. Your template is now saved and can be loaded anytime

Creating a Custom Tab

For frequently used templates, you can create a dedicated tab:

  1. Build and name your template
  2. Click üìå Save as New Tab
  3. A new tab will appear with your template name
  4. This tab will have its own file selection area and process button
üí° Power User Tip: Create tabs for different clients, departments, or project types to quickly switch between different folder structures!

Loading a Template

  1. Select a template from the dropdown menu
  2. Click Load
  3. The template structure will appear in the builder
  4. You can modify it before processing

üéØ Best Practices

Naming Conventions

  • Avoid special characters that aren't allowed in folder names: < > : " | ? *
  • Use underscores or hyphens instead of spaces for better compatibility
  • Keep folder names reasonably short but descriptive

Structure Tips

  • Don't create too many nested levels (3-4 is usually sufficient)
  • Put the most important identifying information at the top level
  • Use date fields for chronological organization
  • Consider how the structure will look with many cases/files

üÜò Troubleshooting

Common Issues

  • Fields not appearing in folders: Make sure the form fields are filled out before processing
  • Can't indent/outdent: Select the item first by clicking on it
  • Template not saving: Enter a template name before clicking save
  • Preview shows errors: Check for invalid characters or missing closing braces in field names
